# SmartSetting2
사용자가 지정한 위치마다 Wi-Fi, Sound Setting, Bluetooth등의 설정을 자동으로 변경한다.</br>
야외에서 불안정한 Wi-Fi 접속상태로 인해 모바일 데이터로 사용중 집에 들어왔을때 실수로,</br>
Wi-Fi On을 하지 않아 등록된 Wi-Fi에 자동연결되지 않아 LTE 데이터를 자신도 모르게 대량소비한 점을 계기로,</br>
개인사용목적으로 개발하였다.</br>
